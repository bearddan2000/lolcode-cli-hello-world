# lolcode-cli-hello-world

## Description
A POC for lolcode programming language.

## Tech stack
- clang
- cmake
- lci
- make
- python

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
